# Car_Sales_Analytics_PowerBI
End-to-end car sales analytics project using Power BI &amp; DAX to analyze YTD sales performance, pricing trends, regional distribution, and dealer-level insights.


 Project Overview

This project focuses on analyzing car sales performance using an end-to-end data analytics approach.
The objective is to uncover sales trends, pricing behavior, regional performance, and dealer-level insights through interactive dashboards.

The final output is a dynamic Power BI dashboard designed for business stakeholders to monitor YTD, MTD, and weekly sales performance.
